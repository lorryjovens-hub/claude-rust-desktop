# Claude Desktop Tauri - 完整 Rust 重构实现计划

## 一、项目概述

根据与 Electron 版本（claude-code-rev）的对比分析，当前 Rust 版本存在以下主要缺失：

1. **核心引擎**：缺少完整的 QueryEngine，当前只有简单的 NativeEngine
2. **权限管理**：完全缺失权限控制系统
3. **MCP系统**：基础实现但缺少高级特性（资源读写、提示模板等）
4. **Skills系统**：只有CRUD操作，缺少执行引擎
5. **记忆系统**：完全缺失 memdir 功能
6. **费用追踪**：功能不完整
7. **钩子系统**：完全缺失
8. **错误处理**：不完善
