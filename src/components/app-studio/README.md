## App Studio — 全流程移动应用开发环境

### 核心理念：Design → Code → Deploy 三阶流水线

```
┌─────────────────────────────────────────────────────────────────┐
│  Design Phase                         Code Phase    Deploy Phase│
│  ┌──────────────┐    ┌──────────────────┐   ┌──────────────┐   │
│  │ Figma-like   │ →  │ Full-stack       │ → │ Expo Build   │   │
│  │ 原型编辑器    │    │ 代码生成 + 联调   │   │ 真机调试      │   │
│  │ 拖拽组件     │    │ React Native     │   │ QR 码安装    │   │
│  │ 点击交互     │    │ API Routes       │   │ 热重载       │   │
│  │ 实时预览     │    │ 数据库 Schema    │   │ 性能监控     │   │
│  └──────┬───────┘    └────────┬─────────┘   └──────┬───────┘   │
│         │                     │                     │          │
│         └────── 一键导出 ─────┴──── Expo Deploy ────┘          │
└─────────────────────────────────────────────────────────────────┘
```
